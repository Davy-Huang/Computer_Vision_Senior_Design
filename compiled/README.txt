This executable converts video data into picture frame data quickly
Note: the video must be in the same directory as the executable

When running the program, you will be prompted for
1. The name of the file, including the extenion
	-this is not case sensitive
2. The name of the folder you would like to store that frame data
	-if the folder already exists, the existing data will NOT be deleted
3. The extension on the image data. 
	-bmp is ideal for positve data
	-jpg is ideal for negetive data
	-Do not enter the '.'
	-For example: 'bmp' is correct. '.bmp' will result in an error
